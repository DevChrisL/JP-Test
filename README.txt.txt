This folder contains all the source code for the JPMC senior porject. To run the code you will need to open the CIS-4910-Senior-Project folder in Visual Studio. You then will need to double click the JP.sln file and press the play button on the top of your screen. We have a separate folder for the job that takes in the new leads every hour from the google form. In the CIS-4910-Senior-Project you will also find the source code for the cron file that sends email out when a new sale has been made. With in the JP file you will find all the HTML, CSS, and C# files. In the Database folder you will find a script to generate all the tables use as well as some excel files that were used to import data. We do not have a file for all the data put into the database. Lastly we have the video presentation of our demo.
